Ejemplos de Multimedia en HTML

Imagen

Usa el elemento <img> para insertar una imagen.

Audio

Usa el elemento <audio> para insertar un archivo de audio.

Video

Usa el elemento <video> para insertar un archivo de video.

Iframe

Usa el elemento <iframe> para insertar contenido de otra fuente, como un video de YouTube.

SVG

Usa el elemento <svg> para insertar gráficos vectoriales escalables.

Canvas

Usa el elemento <canvas> para dibujar gráficos mediante scripts (normalmente JavaScript).

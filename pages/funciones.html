Tipos de Elementos en HTML

<head>

Contiene metadatos/información sobre el documento HTML.

<title>

Define el título del documento, que se muestra en la barra de título del navegador o en la pestaña.

<h1> a <h6>

Define encabezados, siendo <h1> el más importante y <h6> el menos importante.

<p>

Define un párrafo.

<a>

Define un hipervínculo.

<img>

Define una imagen.

<ul> y <ol>

<ul> define una lista desordenada y <ol> define una lista ordenada.

<li>

Define un elemento de lista.

<div>

Define una sección o un contenedor en el documento.

<span>

Define una sección en línea en un documento.

<form>

Define un formulario HTML para la entrada del usuario.

<input>

Define un control de entrada.

<button>

Define un botón clicable.

<table>

Define una tabla.

<tr>, <th>, <td>

<tr> define una fila en una tabla, <th> define un encabezado de celda y <td> define una celda de datos.

<header>, <footer>, <nav>, <section>, <article>, <aside>

Elementos semánticos que definen diferentes partes de una página web.

<nav>

Define una sección de navegación.

<header>

Define un encabezado para un documento o sección.

<footer>

Define un pie de página para un documento o sección.

<section>

Define una sección en un documento.

<article>

Define contenido independiente y autónomo.

<aside>

Define contenido aparte del contenido principal.

<main>

Define el contenido principal del documento.

<figure> y <figcaption>

<figure> define contenido ilustrado y <figcaption> define una leyenda para el contenido.

<blockquote>

Define una sección que es citada de otra fuente.

<code>

Define un fragmento de código.

<pre>

Define texto preformateado.

<em> y <strong>

<em> define texto enfatizado y <strong> define texto importante.

<br>

Define un salto de línea.

<hr>

Define una línea horizontal.

<label>

Define una etiqueta para un control de formulario.

<textarea>

Define un área de texto multilínea.

<select> y <option>

<select> define un menú desplegable y <option> define una opción en el menú.

<iframe>

Define un marco en línea.

<script>

Define un script de cliente.

<style>

Define información de estilo para un documento.

<link>

Define la relación entre un documento y un recurso externo (usualmente una hoja de estilo).

<meta>

Define metadatos sobre un documento HTML.
